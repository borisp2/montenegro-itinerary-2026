# montenegro-itinerary-2026

Private Next.js app for family trip planning in Montenegro, July 18–28, 2026.

## Features
- interactive itinerary day by day
- vacation budget calculator
- interactive Leaflet map with key places
- 3 variants: original, budget, active

## Stack
- Next.js
- React
- React Leaflet
- CSS via `app/globals.css`

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Import the private GitHub repository into Vercel. Vercel supports Next.js natively and deploys GitHub-connected projects directly. [web:3][web:39][web:43]

## Repository setup
Create a private GitHub repository named `montenegro-itinerary-2026`, upload these files, then connect the repository in Vercel. GitHub allows choosing private visibility during repository creation. [web:8][web:11]
